## 🚀 Novedades y Características Principales

### 🖼️ 1. Visor de Fotos Nativo
* **Visualización sin Salir**: Se ha integrado un visor de imágenes nativo directamente en el módulo de Limpiador de Fotos, permitiendo al usuario previsualizar a tamaño completo, ampliar y comparar detalles de imágenes duplicadas o capturas obsoletas antes de decidir su borrado.

### 💾 2. Selección de Unidad en Limpieza Avanzada
* **Limpieza Multi-Unidad**: El módulo de Limpieza Avanzada ahora permite elegir mediante un menú desplegable cuál de las unidades de almacenamiento detectadas en el sistema (C:, D:, E:...) se desea analizar y limpiar.

### 📋 3. Backlog de Desarrollo
* **Seguimiento del Proyecto**: Añadido un archivo físico de control y planificación de tareas (Backlog) bajo el directorio `.agents` para coordinar de forma organizada el roadmap de futuras mejoras de WinCleaner.
