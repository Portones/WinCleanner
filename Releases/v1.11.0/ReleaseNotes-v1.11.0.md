## 🚀 Novedades y Características Principales

### 📦 1. Instalador de Runtimes (Runtime Installer)
* **Instalación Simplificada**: Nuevo módulo integrado que permite la descarga e instalación automatizada de los principales runtimes del sistema (como DirectX, Visual C++ Redistributable, .NET Runtimes) para garantizar la compatibilidad de todos los juegos y aplicaciones en el equipo.

### 🌐 2. Optimizador TCP/IP (TCP Tweaker)
* **Ajustes de Red Avanzados**: Incorporación de una nueva utilidad para optimizar la configuración de red y protocolos TCP/IP, mejorando el ping, la estabilidad de la conexión y las velocidades máximas de transferencia de datos en internet.
